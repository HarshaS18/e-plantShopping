# e-plantShopping

## Paradise Nursery Shopping Application

Welcome to **Paradise Nursery**, an online shopping application for house plants. This React-based e-commerce application allows users to browse and purchase a variety of house plants through an intuitive shopping cart interface.

### Overview

The Paradise Nursery shopping cart application is a full-featured e-commerce solution built with React and Redux. It provides users with a seamless shopping experience for purchasing house plants, featuring:

- **Landing Page**: A welcoming homepage with company information and a "Get Started" button
- **Product Listing Page**: Browse plants organized by categories (e.g., Aromatic Plants, Medicinal Plants) with detailed information including images, descriptions, and prices
- **Shopping Cart**: Manage selected items with quantity controls, item removal, and real-time total calculations
- **Navigation**: Easy navigation between pages with a dynamic cart icon showing the total number of items

### Features

- Dynamic product display with plant cards showing images, names, descriptions, and costs
- Category-based organization of plants
- Add to cart functionality with visual feedback
- Shopping cart management with increment/decrement quantity controls
- Real-time cart total calculations
- Redux-based state management for cart operations
- Responsive navigation bar with cart icon badge

### Technology Stack

- React
- Redux Toolkit
- React Hooks (useState, useEffect)
- Vite
